# Dotfiles

My personal dotfiles.

```
.
├── bash
├── config
│   ├── kitty
│   │   ├── current-theme.conf
│   │   ├── kitty.conf
│   │   └── kitty.conf.bak
│   └── starship.toml
├── README.md
├── resume.sh
└── store.sh
```

## Usage

- Install

```bash
./resume.sh
```

Original dotfiles would be backed up with postfix `.bak`.

- Update

```bash
./store.sh
```
